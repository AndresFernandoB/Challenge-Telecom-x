# Análisis de Evasión de Clientes – Telecomunicaciones

Este proyecto analiza la evasión de clientes (churn) en una empresa de
telefonía e internet utilizando Python y análisis de datos.

## Objetivo
Identificar patrones asociados a la evasión de clientes y variables
con mayor impacto.

## Tecnologías
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Contenido
- Limpieza y transformación de datos
- Análisis exploratorio
- Visualización de churn
- Conclusiones
